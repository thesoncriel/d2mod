D2mod Rune Word Editor

지난번에 만들었던 유니크 에디터를 응용(?) 하여 룬워드 에디터도 만들어 보았습니다...
응용이라고는 하지만, 여러가지 신기술(?)이 들어가서 오늘도 디버그 하다가 쫑냈습니다 ￣∇￣);;
기본적인 사용법은 유니크에디터와 비슷합니다.

실행조건:
* MS Excel 2000 이상 설치된 컴터.
* 사용자는 엑셀 사용법을 조금이라도 알고 있어야 합니다(!)
* 매크로 보안등급을 낮추셔야 합니다.

실행법:
1. 파일을 더블클릭하여 실행
2. 엑셀이 열릴 때 뜨는 창에서 매크로포함(보안수준:보통 일때)을 클릭
3. Runes 시트 선택
4. Rune Word Edit 단추 클릭


화면보기:
가장 우측의 룬워드 리스트 = 룬워드 이름을 리스트화 하여 보여줌. 클릭하면 그에 따른 룬워드 데이터가 아래의 칸들에 표시됨.

RuneWord = 룬워드의 알려진 이름을 표시
Name(Code) = 해당되는 룬워드를 구별하는 코드를 표시
Complete = 체크시, 게임상에서 표시된다는 뜻. 체크 해제시엔 룬워드 데이터를 무시합니다.

Item Type = 룬워드에 필요한 재료 종류를 나열.

그 아래칸(Runes)에는...
룬워드 조합시 나타나는 문자를 표시
가장 긴 쪽이 문장.
그 아래는 필요한 룬을 순서대로 나열함.

Property 1..... = 능력코드(Property Code)와 그에 따른 속성값(Parameter/MinValue/MaxValue)

Rune Info 1..... = 선택한 룬이나 보석들의 능력치를 표시
┣ Weapon Prop = 룬이나 보석을 무기에 장착됐을 시의 옵션
┣ Armor Prop = 룬이나 보석을 방어구에 장착됐을 시의 옵션
┗ Shield prop =  룬이나 보석을 방패에 장착됐을 시의 옵션


버튼들:
Fix!! = 선택한 룬워드의 값을 바꾸고 수정할 때 사용. 한번 바꾸고 난 뒤엔 다시 되돌릴 수 없으므로 백업은 필수입니다.
Clear Data = 새로운 룬워드를 만들고 싶을 때 사용. 화면의 모든 데이터가 사라지고, 작성의 편의를 위해 룬워드 코드에는 자동적으로 이름이 등록됩니다. Complete란도 기본적으로 체크 됩니다.
Add Rune Word = 새로운 룬워드 작성을 마치고 엑셀에 등록할 때 쓰입니다. eol필드는 자동으로 쓰여집니다.




팁:
RuneWord에 찾고싶은 이름을 넣고 엔터를 누르거나 더블클릭하면 그에 해당되는 룬워드를 자동으로 찾아줍니다. 단, 영문자와 띄어쓰기가 정확히 일치해야 합니다...
Item Type에는 일반적으로 사용가능한 아이템형태(Item type)가 마련되어 있습니다. 클릭하여 골라 쓰세요 ^^
Runes에는 룬을 아래 6칸에 맞춰둔 상태에서 더블클릭하면 룬워드가 이어진 형태가 나타납니다. (예: 혼돈-Chaos일 경우: FalOhmUm)




주의:
본 엑셀파일 시트 데이터는 필자가 MOD 제작을 위해 조금씩 변경한 것이 있습니다. 이것을 사용자가 모르거나 무시하고, 작업하던 MOD에 덮어 씌울 경우 차후에 문제가 발생할 소지가 있습니다.
그러므로 필요한 데이터는 자신에게 맞게 조정하거나 기존에 제작하던 MOD 데이터를 이쪽 엑셀에 덮어 씌움으로써 해결할 수 있을 것입니다.(patch_d.mpq파일만 조정하는 사용자에 한함)


개인 MOD제작을 위해 허접하게나마 만들어본 VBA프로그램 입니다.
사용법은 아래와 같습니다.

필요조건:
MS Excel 2000 이상
사용자는 엑셀 사용법을 간단하게 나마 알고 있어야 함.

실행법:
1. UniqueItems.xls 을 연다
2. UniqueItems 시트 중간에 만들어진 단추를 누른다. ※단추이름: Unique Edit


각 부분별 용도:

Item List = 창 우측에 이름들이 나열되어 있는 가장 넓은 공간. 유니크 아이템의 목록이 적혀 있음. 이름순서는 본래 UniqueItems.txt에 있던 그 순서를 그대로 반영한 것.

Name = 아이템 이름
Item Level = 아이템 레벨
Level Req. = 착용 요구 레벨
Item Code = 유니크 아이템의 원본이 되는 노멀아이템(Normal Item)코드
Normal Name = 노멀 아이템의 본래 이름
└우측 = 무기(Weapons) / 방어구(Armor) / 기타(Misc) 구별하는 곳

Carry Option = 소지 한도를 제한하는 곳. No Limit(제한없음) / Only have one(하나만 가질 수 있음)
Cost mult = 용도모름 -_-; 3 or 5의 값이 들어가는데 필요할까봐 넣었음
Cost Add = 유니크 아이템 변환시 추가되는 아이템 값(골드 단위)

Char. Transform = 아이템 색상 바꾸기 - 실재 화면에서 캐릭터가 착용한 상태에서..
Inv. Transform = 아이템 색상 바꾸기 - 인벤토리내의 아이템 색상..
Inv. Image = 아이템 이미지 바꾸기 - 인벤토리내의 아이템 형태에 해당됨. 자세한건 따로 알아보세요 ^^;;

Property 1..... = 아이템의 능력치를 추가 / 변경 / 삭제하는 곳.
좌측부터
Property Code(프로퍼티 코드) / Parameter(파라메터-매개변수) / Min Value(최소값) / Max Value(최대값)
각 종류별로 1~12까지 있습니다. 즉, 아이템 능력 개수는 최고 12개가 최대치 입니다.



버튼들:
Fix!! = 본래 목록에 있던 유니크를 고칠 때 사용. 한번 변경하면 되돌릴 수 없으므로 항상 백업을 해 두세요 =_=
Clear Data = 새로 유니크 목록을 만들 때 사용. 화면에 표시된 데이터가 모두 지워집니다.
Add items = 새로운 유니크 목록을 엑셀파일에 등록할 때 사용. 가장 아래의 데이터로 만들어지고, 우측 리스트가 갱신됩니다. 또한 사소한 필드의 데이터는 자동으로 만듭니다.



팁1 : Name 텍스트박스 부분에 찾는 유니크 이름을 적고 Enter키를 누르거나 더블클릭하면 원하는 유니크로 자동 검색 됩니다. 단, 영문자와 띄어쓰기가 정확히 일치해야 합니다. 대소문자는 안가리는 듯 합니다. (아마도)

팁2 : Item Code에 원하는 아이템 코드를 적으면 Normal Name에 자동적으로 그에 관련된 아이템 이름이 뜨게 됩니다. 반대의 경우도 가능합니다. ^^ 변경 완료시 우측에 아이템의 형태(무기/방어구/기타)가 나타납니다.

팁3 : Char. Transform, Inv. Transform에는 쓸 수 있는 컬러 코드가 리스트로 마련 되어 있습니다. 골라서 쓰세요~

팁4 : Property Code의 화살표를 클릭하시면 기본적으로 쓸 수 있는 모든 코드가 다 나오고, 그에 따른 parameter, MinValue, MaxValue 사용법도 나와 있습니다.

팁5 : property Code에서 Skill code, 혹은 skill tab code를 사용하는 것을 선택하셨을 경우, 그 값을 필요로 하는 곳(Par,Min,Max)에 리스트로 뜨게 되니 참고하십시오.

팁6 : Property Code의 리스트 정렬을 변경하고 싶다면 같은 파일 내의 PropList 시트로 가셔서 살짝 정렬 작업해 주시면 되겠습니다. 기본적으론 Desc 필드에 오름차순입니다.



제작자의 말:
위에 적혀진 기능 이외 다른 것을 수정하는 능력은 없습니다. (예: ladder, rarity,uber 등등)
굳이 수정이 필요하다면 직접 엑셀에서 작업하시길 권합니다. 사실 그리 어렵지도 않습니다 ~_~)/

Item Code와 Normal Name의 코드, 이름을 자동으로 찾아주는 기능엔 약간의 사소한(?) 버그가 있습니다. 무리하게 타이핑하시기 보단, 알려진 코드, 이름을 복사-붙여넣기 하여 편집하시길 권합니다.
또한 몇몇 아이템 이름은, 게임내부의 데이터와 게임외부 데이터(Areat Summit 홈페이지에 등록된)가 서로 다르게 나온 경우도 있습니다. (예: 소서리스 오브 종류 몇가지)
